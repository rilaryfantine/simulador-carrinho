# Simulador de Carrinho Seguidor de Linha

Projeto desenvolvido na disciplina de Técnicas de Programação da FATEC Osasco.

## Sobre o projeto

O sistema simula um carrinho seguidor de linha capaz de percorrer uma pista utilizando sensores virtuais para identificar sua posição e ajustar sua movimentação em tempo real.

## Tecnologias utilizadas

- Python
- Tkinter
- Programação Orientada a Objetos (POO)

## Funcionalidades

- Simulação gráfica em tempo real
- Controle de velocidade
- Pausa e reinício da simulação
- Sensores para acompanhamento da pista
- Interface gráfica interativa

## Aprendizados

Durante o desenvolvimento foram aplicados conceitos de:
- Lógica de programação
- Estruturas de repetição
- Classes e objetos
- Manipulação de coordenadas
- Desenvolvimento de interfaces gráficas
