# Simulador de Carrinho Seguidor de Linha

Projeto desenvolvido na disciplina de Técnicas de Programação da FATEC Osasco.

## Tecnologias utilizadas
- Python
- Tkinter
- Programação Orientada a Objetos

## Funcionalidades
- Simulação de carrinho seguidor de linha
- Controle de velocidade
- Pausa e reinício da simulação
- Interface gráfica interativa

## Objetivo
Aplicar conceitos de lógica de programação, orientação a objetos, animação gráfica e desenvolvimento de interfaces.
